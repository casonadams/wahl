# walh

A neovim theme repo
that is easy on the eyes and
for long hours looking at a screen

- Options for light and dark mode

yes walh does mean what you think it does

**walh** = "winnebago audi lamborghini hatchback"

## Install

### Add plugin

- `casonadams/walh`

### Set and configure theme

```init.vim
set termguicolors
colorscheme walh
```
